# 自动点击器 (AutoClicker)

一个基于 Fabric 开发的 Minecraft 自动点击模组。

## 功能

- 🖱️ 自动攻击准星指向的实体
- 🎲 支持随机点击间隔
- ⌨️ 可自定义快捷键（默认 R 键）
- 💾 配置自动保存

## 指令

| 指令 | 说明     |
|------|--------|
| `/autoclicker` | 显示配置信息 |
| `/autoclicker on` | 开启自动点击 |
| `/autoclicker off` | 关闭自动点击 |
| `/autoclicker fixed <刻>` | 固定间隔模式 |
| `/autoclicker random <最小> <最大>` | 随机间隔模式 |

## 要求

- Minecraft 1.21.11
- Fabric Loader >= 0.18.4
- Fabric API

## 安装

1. 下载最新的 `.jar` 文件
2. 放入 `.minecraft/mods/` 文件夹
3. 启动游戏
